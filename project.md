User selects their sport (tennis, soccer, basketball, swimming, running, CrossFit, etc.)
User records themselves doing 3-4 basic test movements (overhead reach, squat, lunge, rotation)
MediaPipe extracts joint angles from the video in real-time
LLM maps extracted angles against the sport's movement blueprint
Interactive body map lights up: green/yellow/red for each joint based on sport-specific requirements
AI generates a personalized 4-week "return to sport" mobility plan targeting the red and yellow zones
